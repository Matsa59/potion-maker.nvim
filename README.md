# potion-maker.nvim

Be an alchemist on vim.

## Intallation

### Using Packer

```lua
use { 'Matsa59/potion-maker.nvim' }
```

### Bindings

```vim
nmap <leader>mtt :PotionMakerToggleTestFile<CR>
nmap <leader>mts :PotionMakerExecuteTestAtCursor<CR>
nmap <leader>mtS :PotionMakerExecuteTestForCurrentFile<CR>
```
